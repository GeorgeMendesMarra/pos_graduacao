# 🐧 Histórico do Linux

O Linux surgiu como um projeto pessoal e se tornou um dos sistemas mais importantes do mundo.

### 📅 Origem

* Criado em **1991** por Linus Torvalds
* Inicialmente um **kernel** (núcleo do sistema operacional), não um sistema completo
* Inspirado no sistema educacional **MINIX**

### 💡 Evolução

* Código aberto (open source)
* Comunidade mundial começou a contribuir
* Integração com ferramentas do projeto GNU → formando o sistema GNU/Linux

### 🚀 Expansão

* Surgiram distribuições como:

  * Ubuntu
  * Debian
  * Fedora

### 🌍 Hoje

* Presente em:

  * Servidores
  * Supercomputadores
  * Smartphones (Android)
  * Cloud computing

👉 É o sistema mais usado em servidores no mundo.

# 🐡 Histórico do FreeBSD

O FreeBSD tem origem acadêmica e foco em estabilidade e desempenho.

### 📅 Origem

* Baseado no Berkeley Software Distribution (BSD)
* Desenvolvido pela Universidade da Califórnia em Berkeley
* Primeira versão oficial: **1993**

### 💡 Características

* Sistema completo (kernel + ferramentas)
* Licença mais permissiva que o Linux (BSD license)
* Forte foco em:

  * Segurança
  * Performance
  * Redes

### 🚀 Evolução

* Muito usado em:

  * Infraestrutura de internet
  * Firewalls
  * Servidores de alto desempenho

### 🌍 Hoje

* Base de sistemas como:

  * FreeNAS
  * pfSense
* Influenciou sistemas comerciais (ex: macOS usa componentes BSD)

---

# ⚖️ Diferença resumida

| Linux             | FreeBSD                        |
| ----------------- | ------------------------------ |
| Kernel + GNU      | Sistema completo               |
| Licença GPL       | Licença BSD                    |
| Mais popular      | Mais estável (em alguns casos) |
| Grande comunidade | Foco técnico                   |

---

# 🎯 Resumo final

* **Linux** → popular, flexível, comunidade global
* **FreeBSD** → robusto, estável, foco em redes e desempenho
